## 使用

请前往下载 [下载链接](https://github.com/heycn/wallpaper/releases)

目前后端尚未部署到远程服务器，请在本地手动启动后端服务

步骤：

1. `cd /server`
2. `pnpm i`
3. `pnpm start:dev`

我准备了12张壁纸图片，如果想新增，请在 `./server/wallpaper` 目录下自行添加
